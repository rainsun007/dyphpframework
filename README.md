######  DyphpFramework  一个简单易用的MVC开发框架
*此框架为MVC架构的php开发框架，是在工作中逐渐完善而成*

*在不断的优化中参照了一些优秀的框架，同时加入了自己的设计理念，力求轻量、灵活、易用*

*同样采取index.php唯一入口机制，config配制文件控制*


vendors中包含了常用开源类库：PHPExcel,PHPMailer,phpqrcode,PclZip
